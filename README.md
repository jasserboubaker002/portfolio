# 🌐 Mon Portfolio Flutter Web

Portfolio personnel développé avec **Flutter web** pour présenter mes compétences, projets et expériences.

## 🛠️ Technologies utilisées

- **Flutter** (web)
- **Dart**
- **url_launcher**

## ✨ Fonctionnalités

- Design responsive (mobile, tablette, desktop)
- Navigation fluide avec scroll
- Sections : À propos, Projets, Expériences, Contact
- Formulaire de contact
- Liens vers GitHub et LinkedIn

## 🚀 Installation

```bash
git clone https://github.com/jasserboubaker002/portfolio.git
cd portfolio
flutter pub get
flutter run -d chrome
